# 🚀 Aryan Mittal — Full-Stack Developer Portfolio

A high-performance, minimalist, and engineering-focused personal portfolio designed for the modern web. Built with **React 19**, **Vite**, and **Vanilla CSS**, this project showcases a personal brand with a sleek, tech-centric dark mode layout, custom interactive elements, and modern typography.

## ✨ Key Features

- **Personal Brand & Sleek Aesthetic**: Custom "Technical Core" branding featuring a modern dark-mode palette, glassmorphism UI elements, and curated typography (*Syne*, *Bebas Neue*, and *JetBrains Mono*).
- **Interactive UI & Micro-Animations**:
  - Custom responsive dot-and-ring mouse lag cursor tracking.
  - Performance-tuned marquee loops for tech stack display.
  - Scroll-reveal components utilizing the browser's `IntersectionObserver` API.
- **Engineering Mindset**: Dedicated sections highlighting structured thinking, team collaboration, systems scaling, and clean code craft.
- **Lifestyle Section**: A dashboard-like "Beyond Code" view showcasing personal interests like tech exploration, street photography, lo-fi beats, and movies.
- **100% Responsive Design**: Structured with CSS grid systems and fluid layout guidelines to look premium across mobile devices, tablets, and ultra-wide screens.

## 🚀 Getting Started

### Prerequisites

- **Node.js**: `v20.0.0` or higher
- **NPM**: `v10.0.0` or higher

### Installation

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/heyaryanmittal/Portfolio.git
   cd Portfolio
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run the Development Server**:
   Runs the Vite app locally:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   Compiles and minifies assets for production deployment:
   ```bash
   npm run build
   ```

5. **Preview Production Build Locally**:
   ```bash
   npm run preview
   ```
